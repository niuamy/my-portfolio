Projects

Tag Cloud Generator

This was one of my favorite class projects. The objective was to use standard Java to read in a file and output a tag cloud using HTML/CSS, where the size of each word was dependent on the amount of times it appeared in the file

GPA Calculator

This calculator has a dual purpose. One is to allow the user to input their courses and current letter grades to calculate their cumulative GPA. Another feature lets the user calculate their current grade for a given course.

MySite

This is one of my first attempts at a personal website from scratch. It mainly served as a practice project for the purpose of learning Python and HTML/CSS in Flask.
